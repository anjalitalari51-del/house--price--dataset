# 🏠 House Price Prediction using Machine Learning

This project predicts house prices based on various features such as area, number of bedrooms, bathrooms, stories, parking, and more using Machine Learning algorithms in Python.

## 📌 Project Overview
The goal of this project is to build a machine learning model that can accurately predict house prices from given housing data. This helps in understanding how different features affect property prices.

## 📂 Dataset
The dataset contains housing information including:

- Area
- Bedrooms
- Bathrooms
- Stories
- Parking
- Main Road Access
- Air Conditioning
- Furnishing Status
- Price (Target Variable)

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## 📊 Machine Learning Workflow
1. Data Collection
2. Data Preprocessing
3. Handling Categorical Values
4. Train-Test Split
5. Model Training
6. Prediction
7. Model Evaluation

## 🤖 Algorithm Used
- Linear Regression

## 📈 Model Performance
The model was evaluated using:
- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- R² Score
